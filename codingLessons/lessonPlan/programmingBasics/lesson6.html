Tuples

Tuples are enclosed in round brackets or parentheses. You can store different data types in the same tuple. An example is joe_biden = ("Joe Biden", 1942). This tuple contains the name (a string) and the year of birth (an integer) of the current US President. One thing to note about tuples is that you can't add data to them. Tuples are best used to hold data that you don't plan on changing. If you need to add items then you're better off using a list.

Lists

Lists are enclosed in square brackets. Unlike tuples, you can create a list and then add more items to the list. Let's take a look at a list and a few different ways to add items to it. We'll start with a list of US Presidents. They're ordered by terms. This list goes up to the latest officeholder at the time I was born, Bill Clinton. In 2001, George W. Bush is sworn in. Here our code extends the current list with a smaller list containing his name alone. In 2009, Barack Obama is born in. We can see the list ending with Bush is now incremented by adding Obama to the end. After 8 years, Donald Trump is elected. We can see the presidents list is equal to all the previous items in the list plus Trump at the rightmost slot. In 2021, Joe Biden was inaugurated. This piece of code shows the position he is inserted at which is 45 and the item we want to insert at that position, in this case his name.

Iterables

Lists and tuples are referred to as iterables. This is because you're able to iterate over the items they contain. You can use indexing to pluck out individual items from an iterable. You can perform other operations such as slicing, indexing, reversing, and more. Let's take a look using a smaller list for the sake of ease. 

shopping_list = ['milk', 'eggs', 'cheese', 'bread']
print(shopping_list[0])
shopping_list = shopping_list[::-1]
print(shopping_list)
shopping_list.reverse()
print(shopping_list)
print(shopping_list[-1])
print(shopping_list[0:1])
shopping_list.pop(0)
print(shopping_list)
shopping_list.insert(3, 'chicken nuggets')
print(shopping_list)

bread
['milk']

['eggs', 'cheese', 'bread', 'chicken nuggets']

So one important thing to note about lists is that we count from 0. The list begins with 4 items. Similar to someone looking at the items on their list, milk is the first item. The second line of code shows us how to pluck out a specific item from a list: item number 0 is the first item. Likewise in this list with 4 items, item number 3 is the final item. The next line of code reverses the list. A handy thing to remember for this is [start:stop:step] - you use S three times. We don't specify a starting or stopping point so the whole list is involved in this operation. The number in the step section of the square brackets is -1. This means that we are going backwards by one. If we had used -2 then we would also reverse the list but we would only take every second item. Next we use Python's inbuilt reverse function. By reversing a reversed list, we go back to the original list. Now we find the -1st index. By putting -1 as our index, we look for the last item of the list which is bread. The next operation we perform is slicing. We start at the 1st item (index 0) which is milk up to, but not including, the 2nd item (index 1). This in effect grabs a sub list, or a slice, of the original list. This list contains one item - milk. The next operation we perform on this list is pop. We pass the argument 0 to the function. This removes the item at index 0, which is milk, from the list. Printing the list shows the remaining three items in the list. A real world equivalent is a person putting milk into their trolley and crossing it off their list. The next operation we perform on this list is insert. The argument inserts an item, chicken nuggets, to index 3 which adds it to the end of the list. The real world equivalent is someone realising they forgot an item and adding it to their shopping list after leaving the house. We can print the shopping list and see the four items left to buy.
