Lesson 12 Solution

def remove_names(name_list):
    return [name for name in name_list if name[0] not in ("a", "A")]

print(remove_names(["Aaron", "Annie", "Andrew", "Ben", "Chris", "David", "Elle"]))

Lesson 13 Assignment

This week's assignment involves creating a dictionary with a unique id for each key. The corresponding values for each key should contain a list. The list should contain the name, age, sex, desired age range, desired sex and hobbies of a person. Your dictionary should contain a number of entries - one for each person. The point of this week's assignment is to find possible matches for each person in the dictionary. You should check the ages of each person against each other. If one person has an age range of 30-35 then you should not match them with someone 29 or 36. You should want both partners to be satisfied with their potential partners age. You should also make sure the sexes of the partners align. I'm sure a gay man would feel extremely confused if women were showing up as potential matches.

Bonus ideas:
Can you write code to match bisexual users of this dating app to both men and women rather than one sex?
Can you create a list of people which places potential matches with common hobbies at the start of the list and place potential matches with few or no common hobbies towards the end of the list for each person?
