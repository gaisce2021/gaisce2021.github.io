Strings

How do I create a string in Python?

Strings are enclosed in quotes. They can be double quotes for a "string" or single quotes for a 'string'. You can use either one you're consistent. You must keep double quotes in pairs and single quotes in pairs. Either is fine but using one of each for the same string causes an error. Last week's example showed name = "George Washington" as an example of a string. We assigned the string value "George Washington" to the variable name.

Can I format strings?

Like when typing, you may want to use tabs and jump to a new line. Python has these capabilities too. Simply add \t for tabs and \n for a new line. You must keep \t and \n within the quotation marks.

Are there any rules for what can/cannot be put in a string?

You may need to use an apostrophe in a string. But say you've already used single quotes to let Python know you're using a string. You have two options for this. Change the line to something like sentence = "Mary's cat". Or else you could use sentence = 'Mary\'s cat'. The backslash tells Python that the character after the backslash is a special character.

What is string concatenation?

Concatenating strings means joining two or more strings together. One thing to be conscious of is spacing between words when doing this. Let's look at last week's example name = "George Washington". Let's say we wrote things slightly differently. We may have first_name = "George" and last_name = "Washington". Trying to concatenate these with first_name + last_name will give us GeorgeWashington. That's not what we want. But first_name + " " + last_name will give us George Washington which is exactly what we want.

Code Example

Let's try coding this pancake recipe. I'll copy and paste the text below, so you can try formatting it in Python too. As per usual, you can copy and paste the code from the left-hand side of the page. The right-hand side of shows the output of the code. Remember that \n creates a new line and \t is a tab. The print function will print. Each slash shown after the quotes just means that the string continues on the next line. It's a stylistic choice just so each part of the string is on visible on the screen. Here we have used recipe as the variable name. This is good practice as we are using a meaningful name, recipe as our variable, to store our string which contains a pancake recipe.

PANCAKES

Ingredients

  • 100g plain flour
  • 2 large eggs
  • 300ml milk
  • 1 tbsp sunflower oil plus a little extra for frying
  • Lemon wedges, to serve (optional)
  • Caster sugar, to serve (optional)

Method
Put 100g plain flour, 2 large eggs, 300ml milk, 1 tbsp sunflower or vegetable oil and a pinch of salt into a bowl or large jug, then whisk to a smooth batter.
Set a medium frying pan over a medium heat and carefully wipe it with some oiled kitchen paper. When hot cook your pancakes for 1 min on each side until golden.
Serve with lemon wedges and caster sugar, or your favourite filling.

Inputs

You may want to write a piece of code that asks the user to input a piece of information. Let's look through an easy example of this here. We'll ask the user for their name, capitalise their name and print back their name in all capital letters to them. The input function prompts the person using your program to input a piece of information. This is then saved as a string. Why not try converting your name to all lowercase letters or reverse your name? Hint: you'll need the lower function or to add [::-1] after your name. 

user_name = input("Enter your name: ")
print("You entered %s" % user_name)
print("Your name capitalised is %s" % user_name.upper())

You entered Orla
Your name capitalised is ORLA
