Getting Started

This course uses Python as a programming language to learn. Python is a beginner-friendly language with lots of capabilities. Today's lesson involves getting set up and making sure you're comfortable using an IDE before we begin the actual programming. There's no programming involved in today's lesson. This lesson is focused on making sure everything is installed, so you can get moving next week. Following the steps in this lesson will allow subsequent lessons flow much more easily.

Installing Python

Your machine may already have Python installed. If it does, we need to check what version you're running. You can check this from the command prompt on Windows and the terminal on Mac. On Windows, go to the search bar next to the start menu. Type in cmd and the command prompt will come up as the suggested application. On Mac, open up the Launchpad. Type terminal into the searchbar at the top of the screen and open the application. In either operating system, type the command where python and hit enter. If Python is installed on your machine, you should get a file path in return. If not, you can download the latest version of Python here. Now run the command python --version. Make sure that it returns Python 3.7 or a more recent version. If you're running an older version, you can install a newer version from above.

Downloading an IDE

While you can create Python scripts from the command line, using an IDE is a much more user-friendly way of learning to code. An IDE is an integrated development environment. An IDE can let you know if you have poor syntax with red lines. It can also reformat your code to make it a little easier on the eye. Sometimes they give small suggestions to improve your code. There are many IDEs out there; for the purposes of this course download a free one. My IDE of choice is PyCharm. This comes in a free community edition or a paid professional edition. Note that students with a valid email address from their place of study can get the professional edition for free. Another popular IDE is VSCode which allows you to use numerous languages within the same application. If you're more interested in graphs and numbers, then Jupyter Notebook is a great option.

Questions

Why Python instead of another language like Java?

Python is the most popular language according to Statistics Times. Python has many packages for various applications; some are more mathematical, others are more data oriented, some are for visualisation. Python is popular, regularly updated and has a large online community. Packages are well documented and there are many YouTube videos that can give you tutorials for when you get stuck.

What is a package?

A package can be imported onto your laptop. This is code that someone else has written that you can use in your own projects. This is as opposed to writing your own code, you can use well maintained and purpose built libraries to save time. Some libraries such as math are inbuilt libraries. Others are created by companies such as Tableau Server Client are created for use with specific things like Tableau Server in this instance. You will need to download any library you want to use that isn't included in Python's inbuilt library. Some included packages are time and os.

Where can I get help?

There are a number of places you can find help online with coding. Here's a list.

  • Stack Overflow - a programming forum.
  • r/programming - a subreddit.
  • YouTube - lots of programmers upload tutorials here.
  • Documentation - libraries are documented to explain how to use their code.
  • GitHub - a handy way to learn about coding is to look at what someone else did and learn from it.
