Questions

What can I use APIs and requests for?

We generally use them to complete specific applications. We can retrieve data, parse through the data and present the information we are interested in a user-friendly and easy-to-read format.

What projects can you work on using APIs?

You can work on a number of different projects using APIs. We'll start with easier ones that have free API keys so you can practise with any financial worries. You can work on projects relating to the weather, news, movies, games and even more complicated projects which use Google Maps API.

What happens when a server is down?

Unfortunately when a server is down, we cannot retrieve the data we want to work with. This means that any code that you have written after trying to connect to a server will not work. You will need to build some error checking into your code so that the program won't crash if the server is unavailable when you run the code.

Code Examples

Now that we understand what an API is, how JSON data is formatted and a little bit more about HTTP requests; we can finally start to look at using APIs. We are going to use the Pokémon API in today's example. Read the code below and see if you know what the code is trying to do. The output is on the right and the code which you run in your IDE is on the left. 

import requests

url = "https://pokeapi.co/api/v2/pokemon/"
params = {"limit": "10"}
res = requests.request("GET", url=url, params=params)
data = res.json()
for pokemon in data["results"]:
    print(pokemon["name"])

Now that you've had a chance to look at the code, let's see what's happening. The first line is a standard import statement which allows us to use the requests library. The URL is our endpoint. This is where we are retrieving our data from. We have a variable called params which is a parameter to limit the number of Pokémon returned to 10 in our example. We run the requests.request function and use "GET" as one of the parameters. This is equivalent to our requests.get function from last week. We convert the output of the command from JSON format to a dictionary so we can work with the data. We use "results" as an index to pluck the specific results section out of the retrieved data. We then index by "name" within our for loop and print the names of each Pokémon one by one.
