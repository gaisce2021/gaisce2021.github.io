Questions

What is a CSV file?

CSV stands for comma separated values. Excel files can be downloaded in this format. Each piece of data is separated by a comma.

How do I read a CSV file?

You can use the Pandas library to read in the file. I will provide the line of code needed below. We can save this as a variable within our file and work with the data from there.

Is it ok to look at a quick visualisation before modifying a file?

This can be a good way to understand what type of data we're working with. It can also help you see which data is missing that you may miss when working on a CSV file.

Code Examples

You spent last week cleaning up a file and preparing it for visualisation. Perhaps you felt the data provided was sufficient, and you knew straight away what data you wanted to display on a chart. Now let's use code to read the file and save it as a variable. 

import pandas as pd
electionData = pd.read_csv('us_2020_election.csv')

The Pandas library is popular library for data analysis in Python. It can be used to create graphs similar to Matplotlib, but we will use Plotly instead in this section. I find Plotly a little bit more interactive than Pandas. You can hover over graphs and this will display a label. Pandas can be fantastic for manipulating data, so we can use the two libraries in conjunction. 

import plotly.express as px
import plotly.graph_objects as go

Plotly offers programmers numerous graphs. Why not click here and look at the graphs you can use? Look for a graph which you think suits the election data we are focusing on and read the documentation. You can copy and paste the code in their examples and rework it to create a visualisation you feel happy with. You may also wish to make more than one visualisation.

Lesson 50 Solution

This week we will focus on using Plotly to visualise the CSV file. You may use my version of the file. You can also use your version of the file. Take some time to consider what you want to display and why. Read the tips below and see if you can take some inspiration before we move on to the visualisation next week.

  • What are you trying to convey?
  • Do you want to show predicted outcomes in the lead up to the 2020 election and compare it to the results when voters went to the polls?
  • Do you want to show a shift from data in the 2016 election to the more recent 2020 election?
  • Do you want to focus on popular vote or the electoral college?
  • Can you use size, shape or colour to emphasise any data?
  • What colours make sense in the context of US elections and why?
  • Can this data be best shown using trend lines, bar charts, pie charts or a geographic map?
  • How should or could you label the graph?
  • Would someone who casually follows US politics be able to understand the chart you produce?
