Code Examples

Calculus typically involves differentiation and integration. We will start by looking at an example of differentiation as this is usually taught to students first in school. Read through the example below and the code output. After that I'll provide a little paragraph explaining it for you. 

from sympy import *
x = Symbol('x')
y = x**2 + 1
yprime = y.diff(x)
print(yprime)
f = lambdify(x, yprime, 'numpy')
print(f(2))





2*x

4

The first line imports everything within the Sympy library. We designate the letter X as a mathematical symbol and save that as a variable. We create a variable called y and assign its value as being a mathematical equation. This equation is x2 + 1. We create a variable called yprime. This is the result of differentiation the function y. This function is differentiated using a special Sympy function. We print the differentiated function which is 2x. We then use lambdify function which can be used to calculate numerical values quickly. Our variable is x, the function is yprime and the 'numpy' parameter tells the lambdify function that we want to use the NumPy library. We then run print(f(2)) which substitutes in x=2 and gives us an output of 2*2 which is 4.

Let's try integrating 2x and getting x2 + 1. Remember we need a constant of integration, denoted as C, to get that + 1 at the end of our equation. The code here should be self-explanatory as integration is the reverse process of differentiation. 

from sympy import *
x = symbols('x')
yprime = 2*x
c = 1
y = integrate(yprime, x) + c
print(y)
f = lambdify(x, y, 'numpy')
print(f(2))
