Questions

What is an API?

An API is an Application Programming Interface. It works by passing request and response messages between a server and a client.

Can you give me a real life example of an API?

Sure, think about the last time you visited a restaurant. You are the client in this situation. You sit at a table. You look at the available dishes on the menu. You decide to order the chicken curry at this restaurant. The waitress approaches your table. You request your meal from her. The waitress writes this down in her notepad. She walks to the kitchen. The chef asks her what your response was. She instructs him to make you a bowl of chicken curry. He cooks your meal in the kitchen. Once he is finished cooking, she will return to your table with the meal that you have requested. The waitress acts as the API which goes between the patrons of the restaurant and the kitchen staff who represent a client and a server.

How do I do this using code?

The first step is to look at the requests library for Python. Once we understand how to use this, we can learn how to use various libraries and packages to complete more specific tasks and applications.

Code Examples

Before we look at some code snippets, let's look at JSON and HTTP requests. This will help you to understand what we are dealing with before we dive into the Python requests library.

JSON

JSON stands for JavaScript Object Notation. Along with other formats like GSON and XML, JSON is a popular format that is mainly used for interchanging between servers and clients. It allows both you and the company whose API you want to work with store information in an organized and easy-to-access manner and transmit data using HTTP requests. JSON is formatted similarly to a Python dictionary. Let's take a look at an easy example below. 

import json

abrahamLincoln = {
  "name": "Abraham Lincoln",
  "born": "12/02/1809",
  "died": "15/04/1865",
  "presidency": {
    "term_start": "04/03/1861",
    "term_end": "15/04/1865"
  },
  "known_for": ["Emancipation Proclamation",
                "Leading the US through its Civil War"]
}
print(abrahamLincoln["name"])

abrahamLincolnJson = json.dumps(abrahamLincoln)
print(abrahamLincolnJson)

dictionary[key] retrieves the value

HTTP Requests

HTTP stands for Hypertext Transfer Protocol. This protocol is used for transferring data over the internet. It tells web servers and browsers what actions should be taken in response to various commands. These commands include GET, POST, PUT and DELETE are four of the most common commands which you can use.

When you enter a URL of a particular website in your browser, this sends a command to a web server. It asks the server to fetch and transmit the requested web page. You can receive various codes depending on whether the web server exists or not. You may also receive different codes if there is an issue with connectivity on your end or if their server has an issue. 

import requests

response = requests.get('http://www.google.com')
response2 = requests.get('http://www.amazon.ie')
print(response.status_code)
print(response2.status_code)





200
503

Google gives us a 200 response code. This is because we were able to connect to a website which does exist. Amazon returns a 503 response code as they do not currently have an Irish website. You would have better luck trying the German version of the website if you wanted to receive a 200 code. 200 means "OK" whereas a 503 response is status code is "Service Unavailable".
