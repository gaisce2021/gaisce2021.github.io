Code Examples

This will be a more low/no code lesson looking through documentation we can use from the three libraries we have looked at. This will cover some of the main/most popular functions in the three libraries of this section. You may wish to cover these within your own personal projects.

NumPy

  • numpy.sum - This function returns the sum of an array of elements.
  • numpy.sqrt - This function returns the square-root of a single number. It also returns the square-root of an array element wise; this will return [2, 3] if the array you use as an input is [4, 9].
  • numpy.dot - This function returns the dot product of two arrays. Finding the dot product of [4, 9] and [1, 2] will give you (4*1) + (9*2) = 4 + 18 = 22 which is a single number. For 2D vectors, it is equivalent to matrix multiplication. The output of this command will give you a matrix. For the sake of those who have not studied linear algebra, I will not go into more detail here.
  • numpy.pi - This is the constant pi. You may wish to use this in calculations relating to the area or a circle or the volume of a cylinder.

Sympy

  • sympy.diff - This function should look familiar. It is used to calculate the derivative of an expression. Did you know that we can differentiate more than once in one go? If we have look at x3 and differentiate it thrice we will end up with 6. This is because x3 will become 3x2 when differentiated once, which becomes 6x when differentiated twice and finally plain old 6 when differentiated again. Don't believe me? Try running print(diff(x**3, x, 3)) and see the result for yourself.
  • sympy.integrate - We have also looked at integrals before. Did you know we can evaluate an integral from here? We enter our equation, the variable we wish to integrate the equation with respect to, as well as the limits within brackets. Try running print(integrate(cos(x), (x, 0, 1))). A cosine function will return a sine function when you integrate it. We use 0 and 1 as the limits here and this will return sin(1) when we evaluate it.
  • sympy.simplify - This function is used to simplify an expression as the name might suggest. If you add a line equation = 2*x + 5*x and then run the simplify command, the expression will change from 2x + 5x to 7x which is more readable.

Matplotlib

Colours make a graph pop and display some information quickly and easily. The Matplotlib library allows you to use more basic inbuilt colours as well as more specific CSS colours. We will look at the basic colours here.

  • b - Blue
  • g - Green
  • r - Red
  • c - Cyan
  • m - Magenta
  • y - Yellow
  • k - Black
  • w - White

Shapes are another way of making a scatter plot pop. We've already added triangles to one graph. Let's see what we can add to line and scatter plots to make one point of information stand out. We will look at a subset of shapes but you can find the full list here.

  • . - point
  • o - circle
  • ^ - triangle
  • * - star
  • s - square

Lesson 46 Solution

  1. Create a number of x values using the linspace function.
  2. Create a function y = sin x.
  3. Lambdify your function.
  4. Plot this in the colour yellow.
  5. Find the minimum and maximum points on this graph. (Hint: Differentiate y)
  6. Plot these using a purple triangle for each point.
  7. Add a title to this graph.
import numpy as np
import matplotlib.pyplot as plt
from sympy import *

x = symbols('x')
y = sin(x)
y_prime = diff(y, x)

f = lambdify(x, y, modules='numpy')
f_prime = lambdify(x, y_prime, modules='numpy')

x_values = np.linspace(-2 * np.pi, 2 * np.pi, 400)
y_values = f(x_values)

max_points_x = [float(x_val)
            for x_val in solve(y_prime, x)]
max_points_y = [f(x_val)
            for x_val in max_points_x]

plt.plot(x_values, y_values,
            color='yellow', label='f(x) = sin(x)')
plt.plot(max_points_x, max_points_y,
            marker='^', color='purple',
            linestyle='None', label='Max Points')

plt.title('Plot of f(x) = sin(x) and its maximum points')
plt.xlabel('x')
plt.ylabel('y')

plt.legend()
plt.grid(True)

plt.show()
