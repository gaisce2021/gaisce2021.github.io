Questions

Can this text based game be modified into something with graphics?

Probably but we'll deal with a simple graphics based game in future lessons. Now we're focusing on a text based implementation of one of the more popular games in the last 10-15 years to understand some object-oriented programming concepts.

When will we look at a graphics based game?

We will begin that section of this course from lesson 23. If you're less interested in text based games and feel very comfortable with the principles of OOP, feel free to move ahead to begin building a graphics based game.

What will we look at today?

We have created a Character class as a base class. We have created a subclass to inherit from Character for each race within Skyrim. We have created a Skills class which returns a list of length 18. The numbers within the list fall between 15 and 100. Each index represents a different skill such as illusion or smithing.

Today we will review code from last week's assignment. Then we will build a Weapon class. These weapons can be one-handed or two-handed. You can allow your character hold a one-handed weapon in their right or left hand. The character can hold a two-handed weapon using both hands.

Last week I asked you to build 10 classes to extend Character. Here is one possible solution to this task. Remember if you wrote a Character class a little differently, your solution may look different which is perfectly fine. This solution includes the default skill lists for each character. 

class Altmer(Character):
    def __init__(self, name, magicka, health, stamina):
        super().__init__(name, 1, "Altmer", magicka, health, stamina)
        self.skills = altmerSkills

Lesson 20 Assignment

Task 1: Build a class called Weapon that can be used as a base class. This class needs a number of different attributes. These include name, material, weight, value, and damage. Add some methods such as hit and block.

Task 2: Build a two separate child classes of Weapon. You can name these OneHanded and TwoHanded. You can build subclasses of these again based on the Skyrim weapons page by following the link here. Examples of subclasses include Dagger as a child of OneHanded and Battleaxe as a subclass of TwoHanded.
