What is Polymorphism?

Poly comes from the Greek word for many. The morphism part of the word comes from the Greek for shapes. So what is something that is many shaped? In programming this can manifest in the functions that you write. You can create a base class such as Animal. Then you can create two classes such as Pig and Cow which inherit the attributes and methods from Animal. So far so good, right? We covered this in lesson 15. Pig and cows are both animals. Nothing new so far. Legs? 4. That's the same. Now let's say we have a method such as make_noise for the animal class. Now let's see polymorphism in action. A cow says "moo" and a pig says "oink". So we'll need to print something slightly different for both animals.

Why would I need to use polymorphism?

Polymorphism can be used in parent classes when you have a method that can be used in each of the child classes. Having a placeholder method ensures that your code won't crash. When you change the code in a child class, this means you are overriding previously written code. This means that the output you get, like in the method in the animal example, will be more specific for each use case.

Code Examples

Today's coding section will focus on the example mentioned above in the cow and pig section. We will create an animal class and then extend them using the cow and pig classes. We will then create list and loop through all the animals on our farm. This loop will ask the animals to talk. We will then see what each animal has to say. 

from abc import ABC, abstractmethod


class Animal(ABC):
    def __init__(self, legs):
        self.legs = legs

    @abstractmethod
    def make_noise(self):
        raise NotImplementedError("Subclass must implement this method")


class Cow(Animal):
    def __init__(self, legs=4):
        super().__init__(4)

    def make_noise(self):
        return "Moo"


class Pig(Animal):
    def __init__(self, legs=4):
        super().__init__(legs)

    def make_noise(self):
        return "Oink"


cow = Cow()
pig = Pig()
barn = [cow, pig]
for animal in barn:
    print(animal.make_noise())
