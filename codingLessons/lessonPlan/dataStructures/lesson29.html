Questions

What data structure will we look at today?

Today we will look at the queue. This data structure is easy to visualise. Think of the times you've done your grocery shopping; when you pay for your food you're standing in a queue.

How are the queue and the stack different?

The main difference between a stack and a queue is the order in which elements are added and removed. In a stack, the last element added is the first element to be removed (Last In First Out - LIFO). In a queue, the first element added is the first element to be removed (First In First Out - FIFO). Items are removed in the order that they are added within the queue.

What methods are in the queue?

We will focus on five methods. Similar to last time, we have a method to see check if the queue is empty and to see what is at the front of the queue. We will use slightly different methods to add and remove items from the queue as the data structure is FIFO as opposed to the LIFO stack. We also have a size method to check the length of a queue.

  • Enqueue: Add an element to the end of the queue.
  • Dequeue: Remove an element from the front of the queue.
  • IsEmpty: Check if the queue is empty.
  • Peek: Get the value of the front of the queue without removing it.
  • Size: How long is the queue?

Code Examples

Let's assume you own an ice cream shop. Your customers line up. You serve the person closest to the counter. The last person in the queue is the last person to be served until another person joins after them. We can create a Customer class to hold the name and order of the person. 

class Customer:
    def __init__(self, name, order):
        self.name = name
        self.order = order

    def get_order(self):
        return f"{self.name} ordered {self.order}"

Our next task is to create the queue. This will be array based so make sure that is in the constructor. The IsEmpty and size functions will be the easiest to create. Take your time trying to implement the Queue class and then look at my solution below. 

class Queue:
    def __init__(self):
        self.queue = []

    def enqueue(self, item):
        self.queue.append(item)

    def dequeue(self):
        if not self.isEmpty():
            return self.queue.pop(0)

    def isEmpty(self):
        return len(self.queue) == 0

    def peek(self):
        if not self.isEmpty():
            return self.queue[0]

    def size(self):
        return len(self.queue)
