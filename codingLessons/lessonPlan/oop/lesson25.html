What can we do with the canvas from last week?

So far we have learned how to create a canvas, change the colour, change the name and draw shapes on it. Our next task involves making the game more interactive. Now that we know how to draw shapes, we need to make them clickable.

How do we make shapes clickable?

That's the topic of today's lesson. We will look at how to add a type of event listener; the event in this instance is the user clicking with a mouse/trackpad or tapping their phone screen with their finger. Once we have added the code to make a shape clickable, we will add a counter which will update once a shape is clicked. We can use this to track the number of shapes clicked.

Code Examples

Now let's make last week's blue circle example interactive. We will add the lines of code necessary to pre-existing code. After looking at the example below we can then talk through the code which has been added. Amendments to code will be in bold text to clarify what has been updated since last week. 

import tkinter as tk

class Counter:
    def __init__(self):
        self.count = 0

    def increment(self):
        self.count += 1

    def get_count(self):
        return self.count

The first thing we did was to create a counter class. This begins at 0 by default and increases by 1 for each shape you click. Simple enough? Let's look at amending the code some more from last week to implement a working shape counter. 

class Circle:
    def __init__(self, x, y, radius, canvas):
        self.x = x
        self.y = y
        self.radius = radius
        self.canvas = canvas

    def draw(self):
        shape_id = self.canvas.create_oval(self.x - self.radius, self.y - self.radius,
                                           self.x + self.radius, self.y + self.radius,
                                           fill="blue")
        self.canvas.tag_bind(shape_id, '<Button-1>', self.on_click)

    def on_click(self, event):
        counter.increment()
        self.canvas.delete(tk.CURRENT)
        counter_label.config(text="Counter: {}".format(counter.get_count()))

We have now modified the Circle class. The on_click function will increase the counter's value. It then deletes the shape you have clicked on before updating the counter label with the new value. This new value is reflected in a label at the bottom of the screen. 

window = tk.Tk()
window.title("Clickable Circles")

canvas = tk.Canvas(window, width=400, height=400)
canvas.pack()

counter = Counter()

counter_label = tk.Label(window, text="Counter: {}".format(counter.get_count()))
counter_label.pack()

circle1 = Circle(100, 100, 50, canvas)
circle1.draw()
circle2 = Circle(200, 200, 80, canvas)
circle2.draw()

window.mainloop()

We begin by creating a window. We give the window a title. We then create the canvas where the shapes will be drawn. We create an instance of the counter class and add a label section to the bottom of the window. The label function is used to hold text to be displayed in our mini-game. We use the string formatting function which is in built to Python to display the counter which is currently set to 0. We then create and draw two circles from last week. We run the final line of code which launches the window on the screen to create our little game.

Lesson 24 Solution

import tkinter as tk


class Shape:
    def __init__(self, color):
        self.color = color

    def draw(self, canvas):
        pass


class Square(Shape):
    def __init__(self, color, x, y, side_length):
        super().__init__(color)
        self.x = x
        self.y = y
        self.side_length = side_length

    def draw(self, canvas):
        canvas.create_rectangle(self.x, self.y,
                                self.x + self.side_length,
                                self.y + self.side_length,
                                fill=self.color)


class Triangle(Shape):
    def __init__(self, color, x1, y1, x2, y2, x3, y3):
        super().__init__(color)
        self.x1 = x1
        self.y1 = y1
        self.x2 = x2
        self.y2 = y2
        self.x3 = x3
        self.y3 = y3

    def draw(self, canvas):
        canvas.create_polygon(self.x1, self.y1,
                              self.x2, self.y2,
                              self.x3, self.y3,
                              fill=self.color)


window = tk.Tk()
window.title("Shapes")
canvas = tk.Canvas(window, width=400, height=400)
canvas.pack()

square1 = Square("#FFD1DC", 50, 50, 100)
square1.draw(canvas)
square2 = Square("#FFD1DC", 200, 200, 80)
square2.draw(canvas)

triangle1 = Triangle("#89997F", 150, 100, 100, 200, 200, 200)
triangle1.draw(canvas)
triangle2 = Triangle("#89997F", 250, 250, 300, 350, 350, 250)
triangle2.draw(canvas)
triangle3 = Triangle("#89997F", 100, 300, 50, 350, 150, 350)
triangle3.draw(canvas)

window.mainloop()
