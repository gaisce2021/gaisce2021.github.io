Questions

What is a text based game?

A text based game is a game without any graphics. People who grew up in the 1980s or 1990s will likely be a little more familiar with these than people who grew up in the 2000s or 2010s.

What kind of game will we be making?

We're going to build a game based on Skyrim. We will create characters and weapons as classes. Today's lesson will involve creating a character class as a blueprint. Next week we will consider all the unique traits of each type of playable character in the game.

Code Example

Characters in Skyrim are named by the player, we need a name attribute. Characters start at level 1. Characters in Skyrim can be one of many races. Please see the below. 

class Character:
    def __init__(self, name, level=1, race, magicka, health, stamina):
        self.name = name
        self.race = race
        self.magicka = magicka
        self.health = health
        self.stamina = stamina
        self.skills = []
        self.right_hand = False
        self.left_hand = False

    def equip_weapon(self, hand, weapon):
        # hand is either "right" or "left"
        # weapon is a Weapon object
        if hand == "right":
            self.right_hand = weapon
        elif hand == "left":
            self.left_hand = weapon

    def unequip_weapon(self, hand):
        # hand is either "right" or "left"
        if hand == "right":
            self.right_hand = False
        elif hand == "left":
            self.left_hand = False

Lesson 18 Solution

from abc import ABC, abstractmethod


class Car(ABC):
    def __init__(self, model, colour):
        self.model = model
        self.colour = colour

    @abstractmethod
    def get_fuel_type(self):
        pass

    @abstractmethod
    def get_num_gears(self):
        pass

class Kia(Car):
    def __init__(self, model, colour, year):
        super().__init__(model, colour)
        self.year = year

    def get_fuel_type(self):
        return "petrol"

    def get_num_gears(self):
        return 5


class Ford(Car):
    def __init__(self, model, colour, year):
        super().__init__(model, colour)
        self.year = year

    def get_fuel_type(self):
        return "diesel"

    def get_num_gears(self):
        return 6


sportage = Kia("Sportage", "Red", 2017)
fiesta = Ford("Fiesta", "Blue", 2018)
print(sportage.get_num_gears())
print(fiesta.get_fuel_type())

Task 1: Create 10 subclasses of Character. These should be entitled Bosmer, etc. based on ten races that a playable character could be in Skyrim. You can find a list of characters in Skyrim by clicking here. These subclasses should extend the character class I've written for you above.

Task 2: Create a class called Skills. This class needs to return a list of length 18. Each number in the list needs to represent the level of the skill. The skills can be found here. You can create once instance of each skill like altmerSkills = Skills([15, 15, 15, 15, 15, 15, 15, 15, 15, 15, 15, 15, 25, 20, 20, 20, 20, 20]). Do this nine more times for each race.

Task 3: Add the skills you've created for each race into the constructor for its character subclass. For example, the Altmer character subclass should have altmerSkills in its constructor because this is what any player who plays as an Altmer will begin the game with.
