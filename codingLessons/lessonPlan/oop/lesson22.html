How can you expand this game?

For anyone following these lessons that enjoys playing Skyrim, you'll know that this is a massive game. I may not need to give someone like you any ideas. However, if you haven't played this game before, or you haven't played it in a few years, then the tips below might be helpful for you.

You could add an inventory weight to each character. Each weapon we create has a designated weight in our classes. Items like this are stored in a character's inventory. This means that you can only hold so many weapons without being over encumbered. This means you cannot travel, and you move much more slowly. This forces players to store items in a chest, drop them or sell them.

You could add an Archery class. Stealth archer, a character with a high sneak skill and a high archery skill, is a common type of character build. You could create a Bow class along with an Arrow class. You can buy arrows, craft them, loot them from chests or pick them up from the ground.

You could create a Potion class. This would allow you to increase the damage to another player by using a potion on a weapon. You could also take potions to increase your health bar. Potions have a weight, so you can decrease the weight of your inventory for each potion you consume. You can also sell potions if you want to use the gold for something else or reduce your carry weight.

You could create an Ingredient class. Each member of this class has four different effects. You can combine three ingredients together to create a potion. The potion making will fail if the three ingredients have no common effects. You will create a potion successfully if the three ingredients do have some effects in common. Creating potions will increase your alchemy skill.

I have definitely missed many aspects of the game but these are a few ideas to get you started. You may wish to expand upon these ideas and make it into a fully fledged game. Maybe you're happy with the smaller game that we have built together over the past few weeks. Either way I hope you have enjoyed this experience.

class Character:
    def __init__(self, name, level, race, magicka, health, stamina):
        self.name = name
        self.level = level
        self.race = race
        self.magicka = magicka
        self.health = health
        self.stamina = stamina
        self.skills = []
        self.right_hand = False
        self.left_hand = False

    def equip_weapon(self, hand, weapon):
        if weapon.hands == 1:
            if hand == "right":
                self.right_hand = weapon
            elif hand == "left":
                self.left_hand = weapon
        elif weapon.hands == 2:
            self.right_hand = weapon
            self.left_hand = weapon

    def unequip_weapon(self, hand):
        if hand == "right":
            if self.right_hand.hands == 1:
                self.right_hand = False
            elif self.right_hand.hands == 2:
                self.right_hand = False
                self.left_hand = False
        elif hand == "left":
            if self.left_hand.hands == 1:
                self.left_hand = False
            elif self.left_hand.hands == 2:
                self.left_hand = False
                self.right_hand = False

    def fight(self, other):
        if self.right_hand:
            self.right_hand.hit()
            other.health -= self.right_hand.damage
        if self.left_hand and self.left_hand != self.right_hand:
            self.left_hand.hit()
            other.health -= self.left_hand.damage
        if other.health <= 0:
            print(f"{other.name} has died.")



class Skills:
    skill_names = ["Illusion", "Conjuration", "Destruction", "Restoration", "Alteration", "Enchanting",
                   "Smithing", "Heavy Armour", "Block", "Two-Handed", "One-Handed", "Archery",
                   "Light Armour", "Sneak", "Lockpicking", "Pickpocket", "Speech", "Alchemy"]

    def __init__(self, skill_values):
        if len(skill_values) != len(self.skill_names):
            raise ValueError("The list of skill values must have 18 elements")
        for value in skill_values:
            if value < 15 or value > 100:
                raise ValueError("The skill values must be between 15 and 100")
        for i in range(len(self.skill_names)):
            setattr(self, self.skill_names[i], skill_values[i])

    def get_skill_values(self):
        result = []
        for name in self.skill_names:
            result.append(getattr(self, name))
        return result

    def set_skill_value(self, skill_name, new_value):
        if skill_name not in self.skill_names:
            raise ValueError("The skill name must be one of the predefined ones")
        if new_value < 15 or new_value > 100:
            raise ValueError("The skill value must be between 15 and 100")
        setattr(self, skill_name, new_value)

class Weapon:
    def __init__(self, name, material, weight, value, damage):
        self.name = name
        self.material = material
        self.weight = weight
        self.value = value
        self.damage = damage

    def hit(self):
        return "You attack using %s which has %s damage." % (self.name, self.damage)

    def block(self):
        return "Attack blocked!"


class OneHanded(Weapon):
    def __init__(self, name, material, weight, value, damage):
        super().__init__(name, material, weight, value, damage)
        self.hands = 1


class TwoHanded(Weapon):
    def __init__(self, name, material, weight, value, damage):
        super().__init__(name, material, weight, value, damage)
        self.hands = 2


class Dagger(OneHanded):
    def __init__(self, name, material, weight, value, damage):
        super().__init__(name, material, weight, value, damage)


class Battleaxe(TwoHanded):
    def __init__(self, name, material, weight, value, damage):
        super().__init__(name, material, weight, value, damage)
