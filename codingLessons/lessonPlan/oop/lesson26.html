Questions

How are we going to transform the game further today?

This week we will add time to the game. We can set a timer by using Python's time library and users can see how many shapes they can click in a specified window such as 5 seconds.

Where can I take the game after this as we are finishing section two?

You could look into adding more functionality to this game in a number of ways. Allow the user to enter their name. You could add their score to a leaderboard. The leaderboard would need to be amended to reflect the highest scores. This could allow you to compete with family and friends. You could also build a quiz within the tkinter window and add that score to help family/friends who aren't as comfortable clicking quickly using a mouse/trackpad.

Will there be an assignment this week?

No homework this week, lucky you! As we have come to the end of this section of the lessons, take a break and recharge between lessons or review past material you're stuck on. We will return next week to look at a new aspect of Python programming.

What will the next section be about?

The next section will focus on data structures. We have already looked at more primitive data structures such as strings and integers and more complex ones like lists and dictionaries. We will take what we have learned about data types in the last section, Python classes in this section and build more complex data structures in the next section. We will also look at some common use cases for the data structures we learn about in the third section of our course.
