Code Examples

The first thing we will look at is creating different classes to draw different shapes. We can give these classes sensible names such as Circle to make it clear what the class is intended for. We will automatically make all of our circles blue in this example. Now let's talk code.

We will create a constructor in this example. This constructor will take in three values. All values are numeric. It will take in x and y values. These are the coordinates of the centre of the circle. The x value tells you how far left/right it is on the canvas. The y value will give it a place high up or low down on the screen. The next value is the radius. This value decides how "big" your circle is. A small radius will draw a small circle. A large radius will draw a circle with a much bigger area. You may add an option to take in a colour name or a hex value but for the sake of example, we will stick with blue.

Three numbers on their own won't be enough for our example. Now that we have the values of the shape, we need another function to draw it on the canvas. We need to add the following line def draw(self, canvas): to the Circle class. The self refers to the values of the circle that are added by the user. The canvas is where this circle is drawn. We will create a canvas in a later piece of code. The tkinter canvas has an inbuilt create_oval function which we will use here. A circle can be viewed as a specific type of oval which is why we use this function. The four extreme points are entered: lowest x value by subtracting the radius, lowest y value by subtracting the radius, highest x value by adding the radius, highest y value by adding the radius. We also enter fill="blue" to colour in our circle. We will take a look at all this code before adding the canvas below it. 

import tkinter as tk

class Circle:
    def __init__(self, x, y, radius):
        self.x = x
        self.y = y
        self.radius = radius

    def draw(self, canvas):
        canvas.create_oval(self.x - self.radius, self.y - self.radius,
                           self.x + self.radius, self.y + self.radius,
                           fill="blue")

The next step is to create and name the canvas we're working on. We'll use Blue Circles as a working title for the canvas. We can set the dimensions and then use the pack function which is a geometry manager that is used to organize and place widgets within a container, such as a window or a frame.

The next step involves creating and drawing two circles. The first one is centred around the coordinates (100, 100) and the second circle is centred around (200, 200) which is lower down and further to the right on the canvas. The second circle has a radius of 80 pixels making it larger than the first circle which has a radius of 50 pixels. The last step is to add window.mainloop() to run the code. 

window = tk.Tk()
window.title("Blue Circles")
canvas = tk.Canvas(window, width=400, height=400)
canvas.pack()

circle1 = Circle(100, 100, 50)
circle1.draw(canvas)
circle2 = Circle(200, 200, 80)
circle2.draw(canvas)

window.mainloop()
